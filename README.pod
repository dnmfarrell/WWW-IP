=head1 NAME

WWW::IP - Returns your ip address with failsafe mechanism

=head1 VERSION

version 0.01

=head1 SYNOPSIS

    use WWW::IP;

    my $ip = get_ip(); # 54.123.84.6

=head1 EXPORTS

Exports the C<get_ip> function.

=head1 FUNCTIONS

Returns your ip address. Will try a number of services in succession should the 

    use WWW::IP;

    my $ip = get_ip();
